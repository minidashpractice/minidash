# minidash

MiniDash는 Lodash를 경량화하고 간소화 시킨 JavaScript 유틸리티 라이브러리입니다.

## Feature

- 배열, 객체, 문자열을 다루기 위한 사용하기 쉬운 API
- 외부 의존성 없는 경량화

## Install

```bash
npm install minidash
```

## Usage

```js
import {checkedArr} from minidash;

checkedArr([1]); // true
```
